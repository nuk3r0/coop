# GoW Co-Op Mod v3 - Документация

## Обзор изменений

Эта версия включает два ключевых улучшения:

### 1. Автоматический сканер адресов (GameScanner)
Решает проблему зависимости от жестко заданных адресов памяти. Теперь мод может автоматически находить необходимые адреса для любой версии игры.

**Как это работает:**
- Сканирует память процесса на основе сигнатур (patterns)
- Находит локального игрока через множественные паттерны
- Автоматически определяет оффсеты позиции, имени и других данных
- Поддерживает несколько версий игры одновременно

**Файлы:**
- `src/game_scanner/game_scanner.h` - заголовочный файл
- `src/game_scanner/game_scanner.cpp` - реализация сканера

**Использование:**
```cpp
#include "game_scanner/game_scanner.h"

auto result = GameScanner::ScanAll();
if (result.success) {
    // Используем найденные адреса
    g_gow_config.local_player_rva = result.local_player_ptr;
    g_gow_config.pos_x = result.offset_pos_x;
    // и т.д.
}
```

### 2. Современный UI (ImGui)
Полностью переработанный интерфейс с современным дизайном.

**Особенности:**
- Темная тема с акцентными цветами
- Скругленные элементы управления
- Цветовые индикаторы статуса
- Разделенные панели для сущностей и сети
- Улучшенная читаемость и навигация

**Управление:**
- **F1** - Открыть/закрыть главное меню
- **F8** - Открыть/закрыть панель сети

## Решение проблем с UI

Если UI не отображается:

1. **Проверьте лог**: `GoW_CoOp.log` в папке с игрой
2. **Инициализация D3D12**: Убедитесь, что игра использует DirectX 12
3. **Первый запуск**: UI может появиться после первой загрузки в игру
4. **Горячие клавиши**: Нажмите F1 для открытия меню

### Частые проблемы

**UI не виден вообще:**
- Проверьте, что хук Present установлен (смотрите лог)
- Убедитесь, что HWND окна найден корректно

**UI мигает или пропадает:**
- Возможно конфликт с другими оверлеями (Steam, Discord, FRAPS)
- Попробуйте отключить другие оверлеи

**Не работает ввод:**
- Проверьте, что InputManager корректно перехватывает输入

## Расширение сканера для новых версий

Для добавления поддержки новой версии игры:

1. Найдите новые сигнатуры в IDA Pro/Ghidra
2. Добавьте их в массивы `Patterns::LOCAL_PLAYER_PATTERNS` и т.д.
3. Протестируйте с новой версией
4. Обновите логику fallback если нужно

Пример добавления паттерна:
```cpp
inline const char* LOCAL_PLAYER_PATTERNS[] = {
    "48 8B 05 ?? ?? ?? ?? 48 85 C0 74 ?? 48 8B 40 ?? C3",
    "НОВЫЙ_ПАТТЕРН_ЗДЕСЬ",  // Добавьте новую строку
};
```
